Chessbreak


Chessbreak is a clone of chess I decided to do for fun. At first it was an one to one clone of the classic game of chess, but after I finished with the game, I added new pieces to spice up my game. Many of the new pieces I added are silly, but they still work with the game. I decided to make this game, because I was looking for an Object Oriented Programming project. My love for chess definitely swayed me into programming chess other than anything else.

Features

GUI

  • Created with Java JFrame
  • In that JFrame there are 64 JPanels where I add images on top of them for the pieces
  • Menu is created with JMenuBar
I loop through each tile and check if there is an piece there. If there is I get the piece's name and alliance color and add that image to my JPanel.
Code checks all legal moves that the selected piece can make and adds a circle to the JPanel it can move to. This code is my code for my Rook going in the negative y direction. In the project I add this same loop for the positive y direction and twice for the x directions.

Pieces

  • Chessbreak has every piece that was in the classical version of chess plus a few extras
  • Every piece is replacable with a new piece besides the pawn and king
  • The pieces should be balanced enough that the piece is not game breaking
I check where the piece that is selected is and returns a list possible moves. Depending on the current piece's position, there are constraints that may not let it do certain moves. The Move class changes the state the board is in and is the reason the board changes.
Here I go through every piece on the board and get a list of all legal moves every piece on the board can make. Using this list, we can check if a move the user tries to do is a legal move.
This new piece is a queen replacement that has different moves than the queen. This piece is called the Spider and it can only move exactly two tiles away like a donut. This piece is more mobile and can be a great defender in your team and also a great checkmater.
The Spartan is a piece that also moves two tiles away, but this piece can not hit every tile that is two tile away. It only hits tiles that is the opposite color of the tile that piece is on. This piece replaces the queen in the game.
